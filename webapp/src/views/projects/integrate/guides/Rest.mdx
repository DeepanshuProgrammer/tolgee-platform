To read more about Tolgee REST API, please visit [API spec](https://tolgee.io/api).

## Exporting data
To export data via REST API use `{{{apiUrl}}}/v2/projects/export` endpoint.

Don't forget to set `X-API-Key` header to your API key.

### Exporting with CURL

```shell-session
curl "{{{apiUrl}}}/v2/projects/export" \
  -H "X-API-Key: {{{apiKey}}}" \
  --output data.zip
unzip data.zip
rm data.zip
```
